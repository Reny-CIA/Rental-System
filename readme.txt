# Rental System

A **PHP/MySQLi-based House Rental Management System** that helps property managers, landlords, or agencies to efficiently manage rental units, tenants, and payments.  
This system is built to be user-friendly, responsive, and easy to integrate into existing workflows.

---

## 📋 Features

- **User Authentication**
  - Secure login for administrators
  - Session-based authentication to prevent unauthorized access

- **Property Management**
  - Add, edit, and delete property listings
  - Manage rental categories and rates

- **Tenant Management**
  - Register new tenants
  - Track tenant details and history

- **Payment Tracking**
  - Record rent payments
  - Generate payment reports
  - View outstanding balances

- **Reports & Analytics**
  - Filterable reports on tenants and payments
  - Export data for accounting purposes

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (with Bootstrap for UI styling)
- **Backend:** PHP (MySQLi extension)
- **Database:** MySQL
- **Server:** XAMPP (Apache, MySQL, PHP)
- **Version Control:** Git & GitHub (SSH connection)
